## Välkommen till mitt projektarbete i kursen HTML och CSS 2025!
### Projektbeskrivning
I det här projektet ska en responsiv webbplats skapas med HTML och CSS, utan JavaScript. <br>
Webbplatsen ska vara användarvänlig, tillgänglig och responsiv så att den fungera på olika enheter med olika skärmstorlekar.
#### Projektstruktur
README.md and index.html i projektmappens översta nivå<br>
   ./css mapp för styling <br>
   ./images mapp för alla bilder <br>
   ./html mapp för alla webbsidor, utom index.html <br>
   CSS-variabler för färger finns i styles.css<br>
   Styling för header, navigationbar och footer finns i styles.css<br>
   Varje webbsida har sedan en egen css-fil för sitt innehåll i main.<br>

### Layout & responsitivitet
